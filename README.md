
# 🔔 Okul Zili ve Melodi Otomasyonu

Windows Forms ve C# kullanılarak geliştirilmiş, okullar veya kurumlar için özelleştirilebilir zamanlı zil ve melodi çalma otomasyon sistemidir.

> 🎓 **Proje Hikayesi:** Bu proje, lise yıllarımdaki yazılım öğrenme sürecimde geliştirdiğim ilk otomasyon çalışmalarından biridir.

---

## 🚀 Özellikler

* **Zamanlanmış Zil Saatleri:** Günün belirli saatlerine özel otomatik zil ve melodi tetikleme.
* **Medya Oynatıcı Entegrasyonu:** Windows Media Player (`WMPLib`) altyapısı ile ses dosyalarını oynatma.
* **Özelleştirilebilir Melodiler:** Ders giriş, çıkış ve öğretmen zilleri için farklı ses dosyaları atayabilme.
* **Kullanıcı Dostu Arayüz:** Kolay ayarlanabilir ve takibi rahat masaüstü arayüzü.

---

## 🛠️ Kullanılan Teknolojiler

* **Dil:** C#
* **Platform:** .NET / Windows Forms (WinForms)
* **Kütüphaneler:** `AxInterop.WMPLib` / `WMPLib` (Windows Media Player COM)
